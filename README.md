# Integração Continua com GitHub Actions
Projeto com aplicação em Golang usado para estudo de CI/CD com GitHub Actions.

## CI Manual
O arquivo **Makefile** conhecido como solução para automatização de compilação de código fonte escrito em C/C++, aqui é usado para construção de uma pipeline de CI manual.

```bash
# Executa a esteira de CI
make ci

Executa a esteira de CD
make cd

# Executa apenas o lint
make lint

# Executa apenas testes
make test
```
